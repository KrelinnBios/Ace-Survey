# Ace Survey（无性恋社群基本信息调查问卷）

在线访问：<https://ace-survey.pages.dev/>

## 简介

Ace Survey 是一个纯前端、单文件的问卷页面，用于填写和整理无性恋谱系（Ace spectrum）相关的基本信息。页面支持自动保存填写进度，并可将填写结果导出为便于保存或分享的问卷卡片图片。

## 内容与功能

- 问卷填写：围绕无性恋谱系类型、发现经历、身份理解、社群期待等主题提供开放式问题。
- 本地保存：填写内容会自动保存到浏览器本地存储，减少误关闭或刷新造成的内容丢失。
- 图片导出：使用 `html2canvas` 将结果渲染为长图卡片，便于保存、转发或提交。
- 移动端兼容：针对部分移动端 WebView（如 QQ 内置浏览器）和 Firefox 的导出渲染做了兼容处理。

## 使用方式

### 本地使用

直接用浏览器打开 [index.html](./index.html)。

### 线上部署

本项目是静态网页，可部署到 GitHub Pages、Cloudflare Pages、Vercel、Netlify 等静态网站托管平台。

## 内容边界

- 本项目不提供服务器端数据收集功能；填写内容主要保存在访问者自己的浏览器本地存储中。
- 访问者自行填写的问卷回答和导出的个人卡片不属于本仓库预设内容，其使用方式由填写者自行决定。
- 本项目内容仅用于信息整理和社群交流辅助，不构成医疗、心理、法律或其他专业建议。
